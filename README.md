## Assignments

### 1. Constraint Satisfaction Problems (CSP)
Modeling and solving real-world problems as CSPs. Covers constraint propagation techniques (AC-3), backtracking search, and heuristics such as MRV and LCV.

### 2. Hidden Markov Models (HMM)
Implementation and application of Hidden Markov Models for sequence modeling. Topics include the Forward-Backward algorithm, Viterbi decoding, and the Baum-Welch learning algorithm.

### 3. Machine Learning & Classification
Hands-on exploration of supervised learning algorithms including Decision Trees, k-Nearest Neighbors, and Support Vector Machines. Includes model evaluation with cross-validation and performance metrics.

### 4. Minimax & Game Search
Implementation of the Minimax algorithm with Alpha-Beta pruning for two-player zero-sum games. Explores adversarial search trees and game-state evaluation functions.

### 5. Robot Navigation with Bayesian Networks
Probabilistic modeling of a robot's navigation problem using Bayesian Networks. Covers conditional probability tables (CPTs), exact inference, and probabilistic reasoning under uncertainty.

### 6. Neural Networks
Building neural networks from scratch and using deep learning frameworks. Topics include feedforward networks, backpropagation, activation functions, and hyperparameter tuning.

### 7. Reinforcement Learning
Fundamentals of reinforcement learning, including Markov Decision Processes (MDPs), Q-Learning, and policy iteration. Applied to grid-world and classic control environments.

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/javdaninasim/AI_PHW_Spring_2025.git
   cd AI_PHW_Spring_2025
   ```

2. Install dependencies:
   ```bash
   pip install numpy pandas matplotlib seaborn scikit-learn jupyter torch torchvision
   ```

3. Launch Jupyter:
   ```bash
   jupyter notebook
   ```

4. Open the desired assignment folder and run the notebook.
